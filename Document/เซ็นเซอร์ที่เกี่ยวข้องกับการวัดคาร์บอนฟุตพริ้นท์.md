# เซ็นเซอร์ที่เกี่ยวข้องกับการวัดคาร์บอนฟุตพริ้นท์

สำหรับการวัดและติดตามคาร์บอนฟุตพริ้นท์ (Carbon Footprint) อย่างแม่นยำ สามารถแบ่งเซ็นเซอร์ออกเป็น 3 กลุ่มหลักตามหน้าที่ คือ การวัดการปล่อยก๊าซโดยตรง (Scope 1), การวัดการใช้พลังงานทางอ้อม (Scope 2), และตัวบ่งชี้คุณภาพอากาศที่เป็น Proxy ครับ

### 1. กลุ่มวัดการเผาไหม้และการปล่อยก๊าซโดยตรง (Direct Emissions - Scope 1)

กลุ่มนี้สำคัญมากสำหรับโรงงานหรืออาคารที่มีเครื่องกำเนิดไฟฟ้า (Generator) หรือบอยเลอร์ (Boiler) ของตัวเอง

* **Oxygen Sensor (เซ็นเซอร์วัดออกซิเจน):**
    * **หน้าที่:** ใช้ในระบบ "O₂ Trim" ของบอยเลอร์ เพื่อวัดปริมาณออกซิเจนที่เหลือในปล่องไอเสีย[^1]
    * **ความสำคัญกับคาร์บอน:** ช่วยหาจุดสมดุลของการเผาไหม้ (Air-to-Fuel Ratio) ถ้าออกซิเจนเหลือน้อยไปจะเกิดเขม่า (Black Carbon) ถ้าเหลือมากไปแสดงว่าสูญเสียความร้อนทิ้งไปเปล่าๆ การปรับจูนจุดนี้ช่วยลดการใช้เชื้อเพลิงและลดคาร์บอนได้โดยตรง
* **CO₂ Sensor (เซ็นเซอร์วัดคาร์บอนไดออกไซด์):**
    * **หน้าที่:**

1. **ในโรงงาน:** วัดปริมาณก๊าซเรือนกระจกที่ปล่อยออกจากปล่องไอเสีย (Stack Emissions) เพื่อทำรายงานตามกฎหมาย[^2]
2. **ในอาคาร:** ใช้ควบคุมระบบระบายอากาศ (Demand Control Ventilation) ถ้าคนน้อย CO₂ ต่ำ ระบบจะลดการดึงลมร้อนจากภายนอกเข้ามา ช่วยประหยัดไฟแอร์ได้มหาศาล[^3]
* **Temperature Sensor (เซ็นเซอร์วัดอุณหภูมิ):**
    * **ความสำคัญ:** ใช้วัดอุณหภูมิไอเสีย (Stack Temp) ถ้าอุณหภูมิสูงผิดปกติ แสดงว่ามีการถ่ายเทความร้อนที่ไม่ดี (เช่น ท่อสกปรก) ทำให้สิ้นเปลืองเชื้อเพลิงและปล่อยคาร์บอนเกินความจำเป็น


### 2. กลุ่มวัดการใช้พลังงานไฟฟ้า (Indirect Emissions - Scope 2)

เป็นการวัดคาร์บอนที่เกิดจากการผลิตไฟฟ้าที่เราซื้อมาจากสายส่ง

* **Power Meter (มิเตอร์วัดพลังงานไฟฟ้า):**
    * **หน้าที่:** วัดหน่วยการใช้ไฟฟ้า (kWh) อย่างละเอียด
    * **การคำนวณคาร์บอน:** นำค่า kWh ที่ได้ไปคูณกับ **Emission Factor** ของการไฟฟ้าฯ (ค่าสัมประสิทธิ์การปล่อยก๊าซเรือนกระจกของสายส่ง) เพื่อออกมาเป็นปริมาณคาร์บอน (kgCO₂e) เป็นอุปกรณ์พื้นฐานที่สุดสำหรับการทำ Carbon Accounting[^4]
* **Light Sensor (เซ็นเซอร์แสงสว่าง):**
    * **หน้าที่:** ใช้ในระบบ Daylight Harvesting เพื่อหรี่ไฟลงเมื่อมีแสงธรรมชาติเพียงพอ ช่วยลด Peak Load ของระบบไฟฟ้า ซึ่งมักเป็นช่วงที่โรงไฟฟ้าปล่อยคาร์บอนสูงที่สุด


### 3. กลุ่มตัวบ่งชี้และมลพิษร่วม (Proxy \& Co-benefits)

ไม่ได้วัด CO₂ โดยตรง แต่วัดสารที่มักจะถูกปล่อยออกมาพร้อมกัน หรือบ่งชี้ประสิทธิภาพเครื่องจักร

* **PM2.5 Sensor (เซ็นเซอร์ฝุ่นละเอียด):**
    * **ความสำคัญ:** ในบริบทของคาร์บอน ค่า PM2.5 ที่สูงจากการเผาไหม้ มักมีความสัมพันธ์โดยตรงกับ **Black Carbon (เขม่าดำ)** ซึ่งเป็นตัวการสำคัญที่ทำให้โลกร้อน (Short-lived climate pollutant) ค่าที่พุ่งสูงขึ้นมักบ่งบอกถึงเครื่องยนต์ดีเซลหรือการเผาไหม้ที่บกพร่อง[^5][^6]
* **Dust Sensor (เซ็นเซอร์ฝุ่นทั่วไป):**
    * **ความสำคัญ:** ใช้เฝ้าระวังระบบกรอง (Baghouse filters) ในโรงงาน ถ้าระบบกรองตันหรือรั่ว จะทำให้พัดลมดูดอากาศทำงานหนักขึ้นและกินไฟมากขึ้น (เพิ่ม Scope 2)


### สรุปตารางความสัมพันธ์ (ภาษาไทย)

| เซ็นเซอร์ | ข้อมูลที่ได้ | การแปลงเป็นค่าคาร์บอน |
| :-- | :-- | :-- |
| **Power Meter** | kWh (พลังงานไฟฟ้า) | `kWh × Emission Factor ของกริดไฟฟ้า` |
| **O₂ Sensor** | % O₂ (ในไอเสีย) | ใช้จูนเครื่องจักรให้กินน้ำมันน้อยลง (ลด Scope 1) |
| **CO₂ Sensor** | ppm หรือ % | วัดปริมาณก๊าซโดยตรง หรือ ใช้ลดภาระแอร์ในอาคาร |
| **PM2.5** | µg/m³ | ใช้ประเมินเขม่าดำ (Black Carbon) จากเครื่องยนต์ |
