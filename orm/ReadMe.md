### ORM (Object-Relational Mapping) คืออะไร
 - ORM (Object-Relational Mapping) เป็นเทคนิคการเขียนโปรแกรมที่แปลงข้อมูลระหว่างฐานข้อมูลกับภาษาเชิงวัตถุ โดย TypeORM เป็น ORM ที่ได้รับความนิยม สำหรับแอปพลิเคชัน Node.js และ TypeScript ซึ่งช่วยให้นักพัฒนาสามารถทำงานกับฐานข้อมูลได้โดยใช้แนวคิดเชิงวัตถุแทนการเขียน SQL โดยตรง. 


### TypeORM คืออะไร
 - ORM แบบเฉพาะ: TypeORM เป็นไลบรารี ORM ที่ออกแบบมาเพื่อทำงานกับ TypeScript และ JavaScript โดยเฉพาะ.
